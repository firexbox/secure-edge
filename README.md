# secure-edge

这套程序为你打造了一个“阅后即焚、密码上锁”的浏览器安全沙箱，所有的浏览记录、书签和 Cookie 都会被锁在极高强度的加密容器中。
---

## 🛡️ Secure Edge 完整使用指南

### 一、 核心功能概述
Secure Edge 是一个基于本地脚本驱动的隐私保护工具，它主要做两件事：
1. **启动拦截**：没有正确的密码，任何人都无法通过该脚本启动你的专属 Edge 环境。
2. **数据加密**：使用军工级加密工具（VeraCrypt）在你的电脑上创建一个“黑匣子”（`UserData.hc`）。只有在你输入密码且验证通过后，这个黑匣子才会变成一个临时的 `Y:` 盘供 Edge 使用。浏览器一关，`Y:` 盘瞬间消失，不留痕迹。

---

### 二、 首次运行配置（只需做一次）

在开始日常使用前，你需要完成两步初始化配置。请在存放脚本的文件夹中打开终端（或者命令提示符/PowerShell）：

#### 步骤 1：设置浏览器启动密码
这是每次双击启动脚本时，用来验证你身份的密码。
1. 运行命令：
   ```cmd
   .\se.bat --setup-password
   ```
2. 按照提示输入密码并确认。
3. **成功标志**：控制台显示绿色的 `Edge password set successfully!`。此时会在 `config_edge` 文件夹下生成一个隐藏的 `password_edge.enc` 文件。

#### 步骤 2：创建加密黑匣子 (VeraCrypt 容器)
这是用来存放你所有浏览器数据的加密库。
1. 运行命令：
   ```cmd
   .\se.bat --setup-encryption
   ```
2. 脚本会自动加载加密模块，如果这是你第一次运行，它会自动下载便携版 VeraCrypt（无需安装），如果没有下载，到`https://veracrypt.io/zh-cn/Downloads.html`下载便携版，安装至`secure-edge\veracrypt`目录下，并将对应`.exe`文件名中的`-x64`去掉，`VeraCrypt Format.exe`，`VeraCrypt.exe`，`VeraCryptExpander.exe`。
3. 按照提示进行选择：
   * **容量大小**：通常建议选择 **2 (200MB)** 或 **3 (500MB)**，如果你有大量扩展或经常缓存，可以选择自定义更大的容量。如果后续需要增加容量，启动`VeraCrypt.exe`，选择工具→加密卷扩展向导扩展至新的容量。
   * **加密盘密码**：为这个黑匣子设置密码。（💡 **强烈建议**：为了日常使用方便，这里的密码最好和步骤 1 中设置的启动密码保持一致。）
4. **成功标志**：控制台提示创建成功，并在当前目录下生成一个 `UserData.hc` 文件。

---

### 三、 日常使用方法

配置完成后，日常使用非常简单：

1. **启动**：直接双击 **`se.bat`**（或者在命令行运行它）。
2. **解锁**：在弹出的黑色控制台窗口中输入你的密码（输入时密码是不可见的，输完按回车）。
3. **浏览**：验证成功后，脚本会在后台默默将 `UserData.hc` 挂载为虚拟的 `Y:` 盘，并自动打开 Edge 浏览器。此时，你可以像往常一样安全地上网。
4. **安全退出**：当你**彻底关闭** Edge 浏览器后，控制台脚本检测到浏览器进程结束，会自动卸载 `Y:` 盘。此时所有数据重新被封锁，彻底隐身。

---

### 四、 ⚠️ 关键注意事项（必读）

* **切勿丢失密码！** 无论是启动密码还是加密盘密码，一旦遗忘，**绝对没有任何后门可以找回**。你的书签、密码记录将随之永久锁死在黑匣子里。
* **彻底关闭浏览器才能上锁**：
  Edge 有一个“后台运行扩展和应用”的功能。如果在关闭浏览器窗口后，Edge 仍在后台运行，虚拟 `Y:` 盘将无法安全卸载。
  > **建议**：在 Edge 设置中（`edge://settings/system`），关闭**“在 Microsoft Edge 关闭后继续运行后台扩展和应用”**的选项，确保做到“关窗即落锁”。
* **关于杀毒软件**：
  由于这套工具涉及隐藏文件的生成、虚拟盘的挂载以及调用加密软件，部分杀软（如 360、火绒、Windows Defender）可能会将其判定为可疑行为。如果运行失败，请将该文件夹加入杀软的**信任白名单**。
* **数据备份**：
  你所有的秘密都在 **`UserData.hc`** 这个文件里。换电脑或重装系统时，只需要把整个文件夹（特别是这个 `.hc` 文件）拷贝走，在新电脑上运行 `se.bat` 即可无缝恢复。
